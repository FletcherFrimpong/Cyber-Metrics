# PDF Export Guide - Trends Analytics Dashboard

## 🎯 Overview

The Trends Analytics Dashboard now includes a functional **Export Report** button that generates comprehensive PDF reports of your security analytics data.

## 🚀 How to Use

### **Step 1: Navigate to Trends Analytics**
1. Go to the main dashboard
2. Click on "Trends & Analytics" in the navigation
3. The dashboard will load with current security metrics

### **Step 2: Configure Your View**
1. **Analysis Period**: Choose between "Quarterly" or "Yearly" view
2. **Quarter Selection**: If using quarterly view, select the specific quarter (Q1-Q4 2025)
3. **Review Data**: Ensure the dashboard displays the data you want to export

### **Step 3: Export PDF Report**
1. Click the **"Export Report"** button in the top-right corner
2. The button will show "Generating PDF..." while processing
3. A new browser window will open with the formatted report
4. The browser's print dialog will automatically appear
5. **Save as PDF**: In the print dialog, select "Save as PDF" as the destination
6. Choose your save location and filename
7. Click "Save" to download the PDF

### **Trends Analysis**
- **Quarterly Trends**: Attack patterns and cost savings by quarter
- **Yearly Comparison**: Year-over-year performance metrics
- **Growth Rate Analysis**: Percentage changes in security metrics

## 🚨 Troubleshooting

### **Common Issues**

#### **1. Print Dialog Doesn't Appear**
- **Solution**: Check if pop-up blockers are enabled
- **Fix**: Allow pop-ups for the SignalFoundry domain

#### **2. PDF Generation Fails**
- **Solution**: Ensure you have sufficient data for the selected period
- **Fix**: Try selecting a different quarter or yearly view

#### **3. Browser Compatibility Issues**
- **Solution**: Use Chrome or Edge for best compatibility
- **Fix**: Update your browser to the latest version

#### **4. Large File Size**
- **Solution**: The PDF uses efficient formatting
- **Fix**: If issues persist, try exporting smaller time periods

### **Error Messages**

#### **"Failed to generate PDF report"**
- Check your internet connection
- Ensure the dashboard has loaded completely
- Try refreshing the page and exporting again

#### **"No data available for export"**
- Select a different time period
- Ensure the dashboard is displaying data
- Check if the selected quarter has available data

## 📈 Best Practices

### **For Optimal PDF Reports**

1. **Data Selection**
   - Choose time periods with sufficient data
   - Use quarterly view for detailed analysis
   - Use yearly view for executive summaries

2. **Export Timing**
   - Export after dashboard has fully loaded
   - Ensure all charts and metrics are visible
   - Avoid exporting during data refresh

3. **File Management**
   - Use descriptive filenames (e.g., "Q3-2025-Security-Report.pdf")
   - Organize reports by date and quarter
   - Keep reports for historical comparison
